# expresso

A social media website - built using React, React-router-dom, tailwind css, and a mock backend.

## features  
- User Feed
-- all user posts
-- Sorting by Date & Trending
- Create a Post
- Like a post
- Edit & Delete Post
- User Profile
-- Add Avatar
-- Add Bio
-- Add Portfolio URL
- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Authentication
-- Sign-up
-- Login
-- Logout
